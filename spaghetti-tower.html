Spaghetti Tower

Project Description

For this project we had to build a tower with spaghetti and glue. We could only use twenty-five strands.

I thought a tapering tower with a wide base would work best. We did not choose that design because it used too much material. We made a triangular base instead.

Another team used the tapering design and did better than we did. This showed how much the material limit changed our result.

Overall Project Reflection

This project showed me how limits can change a design. If I did it again I would test small versions first but we didn't have any materials to test with.

I also learned that teamwork matters during a build.
